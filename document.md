**Документация приложения по аренде авто**

**Как правильно использовать приложение.**
1. Открыть приложение и пройти регистрацию или войти нажав нужную кнопку (РЕГИСТРАЦИЯ ИЛИ ВОЙТИ)
2. Произведя вход в приложения вы попадаете на главную страницу где будут представлены доступные для аренды транспортные средства и кнопка для их аренды
3. Дальше при оформлению аренды авто проследуйте на другую страницу под названием мои авто там будут ваши арендованные авто и время окончания срока         их аренды
**Как правильно пользовать приложение для (Админа).**
1. Администратор может так же использовать приложение как и обычный пользователь
2. Администратор может блокировать аккаунты обычных пользователей
3. Администратор может ставить машины в стоп лист, стоп лист это приостановка 
аренды авто в приложении
4. Также администратор имеет доступ к базе данных приложения

Тип приложения: база данных / справочник / каталог авто. Целевая аудитория: 
взрослые, семьи, молодые, начинающие водители, любители авто. Платформы: Android / iOS / Web (в планах). Интерфейс: минималистичный, с быстрым поиском нужного авто и его характеристик.

**Преимущества**
